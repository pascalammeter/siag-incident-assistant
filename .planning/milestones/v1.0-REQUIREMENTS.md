## Changes During Development

**Requirements Validated:**
- All 13 requirement groups (F1–F8, NF1–NF5) validated through implementation
- No requirement changes during milestone — scope remained stable

**Decisions Made:**
- Next.js 16 chosen over 15.x for latest features (API-compatible)
- Tailwind v4 with @theme{} pattern reduces config complexity
- Static export prioritized for MVP (no backend complexity)
- localStorage chosen over cookies for PWA compatibility

**Known Gaps:**
- GitHub webhook integration issue (documented, CLI workaround provided)
- Turbopack workspace warning (environmental, non-blocking)
