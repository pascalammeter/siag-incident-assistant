### Task 1: Implement actual soft-delete in IncidentService
**Type:** auto

1. Open `src/api/services/incident.service.ts` (lines 58-73)
2. Replace the entire deleteIncident() method with actual soft-delete logic:
   ```typescript
   static async deleteIncident(id: string) {
     const incident = await prisma.incident.findFirst({
       where: {
         id,
       },
     });

     if (!incident) {
       return null;
     }

     // Soft-delete by setting deletedAt timestamp
     const deleted = await prisma.incident.update({
       where: { id },
       data: {
         deletedAt: new Date(),
       },
     });

     return deleted;
   }
   ```
