## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JSDoc → OpenAPI spec | JSDoc blocks parsed by swagger-jsdoc; malformed blocks could inject invalid OpenAPI |
| /api/swagger endpoint | Public endpoint serves API documentation; no authentication required |
| Curl examples in JSDoc | Examples contain placeholder API keys; ensure NOT hardcoded real keys |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-21-01 | Disclosure | JSDoc curl examples | mitigate | Use placeholder API keys (sk_test_...) in curl examples, never real keys |
| T-21-02 | Tampering | swagger-jsdoc parsing | accept | swagger-jsdoc v6.2.8 is established library with security track record; OpenAPI spec validation in Swagger UI catches syntax errors |
| T-21-03 | Denial | /api/swagger endpoint | accept | Swagger UI is public documentation endpoint; API key auth still required for actual endpoint calls |

</threat_model>
