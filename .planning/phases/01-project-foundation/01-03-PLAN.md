
   ```bash
   git config user.email "79907325+pascalammeter@users.noreply.github.com"
   ```

2. Sicherstellen dass .gitignore folgende Eintraege hat (create-next-app erstellt diese normalerweise):
   - `node_modules/`

   - `out/`
   - `.env*.local`

3. GitHub Repository erstellen via gh CLI:
   ```bash
   gh repo create siag-incident-assistant --public --source=. --remote=origin --push
   ```
   Falls Repo bereits existiert oder remote origin schon gesetzt: nur pushen:
   ```bash
   git push -u origin main
   ```
   HINWEIS: --public damit Vercel Hobby kostenlos deployen kann. Falls private gewuenscht, muss Vercel-Team konfiguriert werden.

4. Pruefen dass Push erfolgreich war:
   ```bash
   gh repo view --web 2>/dev/null || gh repo view
   ```
