  <action>
1. Projekt mit create-next-app erstellen (OHNE --tailwind Flag — installiert sonst v3):
   ```bash
   npx create-next-app@latest . --typescript --eslint --app --src-dir --no-tailwind --import-alias "@/*"
   ```
   Bei Prompt "Use Turbopack": Yes. Bei anderen Prompts: Defaults akzeptieren.
   WICHTIG: Das `.` am Ende, weil wir im Projektverzeichnis sind.

2. Tailwind v4 manuell installieren:
   ```bash
   npm install tailwindcss @tailwindcss/postcss postcss
   ```

3. Falls create-next-app eine `tailwind.config.ts` oder `tailwind.config.js` erstellt hat: LOESCHEN. Tailwind v4 braucht keine JS-Config.

4. `postcss.config.mjs` ersetzen mit:
   ```javascript
   const config = {
     plugins: {
       "@tailwindcss/postcss": {},
     },
   };
   export default config;
   ```
   WICHTIG: Kein `tailwindcss` oder `autoprefixer` Plugin — das sind v3-Plugins. Nur `@tailwindcss/postcss`.

5. `next.config.ts` ersetzen mit:
   ```typescript
   import type { NextConfig } from "next";

   const nextConfig: NextConfig = {
     output: "export",
     images: {
       unoptimized: true,
     },
   };

   export default nextConfig;
   ```

6. `src/app/globals.css` ersetzen mit:
   ```css
   @import "tailwindcss";

   @theme {
     /* SIAG Brand Colors */
     --color-navy: #1a2e4a;
     --color-navy-light: #2a4a6b;
     --color-navy-dark: #0f1e33;
     --color-white: #ffffff;
     --color-lightgray: #f5f7fa;
     --color-amber: #f59e0b;
     --color-amber-light: #fbbf24;
     --color-amber-dark: #d97706;

     /* Font */
     --font-sans: "Inter", ui-sans-serif, system-ui, sans-serif;
   }

   /* Base body styles */
   body {
     font-family: var(--font-sans);
     color: var(--color-navy);
     background-color: var(--color-white);
   }
   ```
   WICHTIG: Keine `@tailwind base/components/utilities` Direktiven — das ist v3-Syntax. Nur `@import "tailwindcss"`.

7. `src/app/page.tsx` mit minimalem Inhalt ersetzen, der Tailwind-Klassen nutzt (zum Testen):
   ```tsx
   export default function Home() {
     return (
       <div className="flex items-center justify-center min-h-[60vh]">
         <div className="text-center">

           </h1>
           <p className="text-navy/60">
             Wizard wird in Phase 2 implementiert.
           </p>
         </div>
       </div>
     );
   }
   ```

8. `src/app/layout.tsx` minimal halten — nur globals.css importieren, html lang="de" setzen, metadata konfigurieren:
   ```tsx
   import type { Metadata } from "next";
   import "./globals.css";

   export const metadata: Metadata = {
     title: "SIAG Incident Management Assistent",
     description: "Incident-Response-Wizard fuer SIAG-Berater",
   };

   export default function RootLayout({
     children,
   }: {
     children: React.ReactNode;
   }) {
     return (
       <html lang="de">
         <body>{children}</body>
       </html>
     );
   }
   ```
   HINWEIS: Layout wird in Plan 02 durch Header/Footer erweitert. Hier nur Minimal-Version.
